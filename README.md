# wookhyung.dev

https://wookhyung.dev

Personal blog built with Next.js.

## Tech Stack

- Next.js 15, React 19, TypeScript
- Tailwind CSS 4, Motion (Framer Motion)
- Contentlayer2, MDX

## Scripts

```bash
pnpm dev     # Development server
pnpm build   # Production build
pnpm start   # Production server
```
